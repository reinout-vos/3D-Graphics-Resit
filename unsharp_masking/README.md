# Runnning the project:
Make sure you have node / npm installed (https://nodejs.org/en).

Then go to the 'unsharp_masking' directory:
```bash
cd unsharp_masking
```

Install the required npm packages:
```bash
npm install
```

Run the project:
```bash
npm run dev
```

You should now be able to see the webpage at:
```
http://localhost:5173/
```
